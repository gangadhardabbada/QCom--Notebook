# 30 Days Quantum Series Notes

> Personal notes and implementations based on the **30 Days Quantum Series** by **PROJECT-Q Community**.

## About

This repository contains my day-wise learning notes, code examples, and practical implementations completed while following the **30 Days Quantum Series** organized by **PROJECT-Q Community**.

The goal of this repository is to document my learning journey and serve as a structured reference for anyone beginning quantum computing. Topics progress from foundational quantum concepts to implementing quantum circuits and algorithms using **Python** and **Qiskit**.
